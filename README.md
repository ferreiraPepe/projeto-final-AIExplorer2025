# Projeto Final AI Explorer 2025
Repositório para guardar o projeto final do curso AI Explorer 2025. Uma jornada focada em construir agentes de IA utilizando como ferramenta o Langflow. 

Maiores informações sobre a construção do projeto, no arquivo pdf presente aqui no repositório.

## Como rodar o projeto?
Para que consiga utilizar o agente de IA, primeiro é necessário que crie uma conta na DataStax. Assim que estiver criada, entre na ferramenta Langflow e lá haverá uma opção "New Flow".
Clique e arraste o arquivo json, anexado aqui no repositório, para dentro do seu flow.

Após se deparar com toda a estrutura já presente na sua tela, você deve inserir suas chaves de API de cada ferramenta (Tavily e Gemini). Com todas as chaves inseridas, só é necessário clicar em "playground" no canto superior direito.
